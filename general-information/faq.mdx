<AccordionGroup>
    <Accordion
        title="What is CinePro?"
        icon="question"
    >
        CinePro is an **open-source movie and TV show scraper API** designed to provide links to media content available on the internet. It scrapes links from various independent sources and is completely free of ads and tracking.
    </Accordion>

    <Accordion
        title="Is CinePro free to use?"
        icon="money-bill"
        
    >
        **Yes, CinePro is completely free to use!**
    </Accordion>

    <Accordion
        title="Is this legal?"
        icon="scale-balanced"
        
    >
        CinePro is a tool for educational purposes only. It does not host or upload any media files. The legality of the content you access through CinePro **depends on the sources and your local laws**. Always ensure you are complying with copyright laws in your jurisdiction.
    </Accordion>

    <Accordion
        title="What quality are the links?"
        icon="film"
    >
        The quality of the links varies depending on the source. CinePro scrapes links from various independent sources, so the quality may differ. Each file has its own quality. You can check the quality of the file in the response.
        We do have providers that host everything up to 4K, but the quality of the files you might get is not guaranteed.
    </Accordion>

    <Accordion
        title="Can I use CinePro for commercial purposes? (e.g, using on a public site that you make money from)"
        icon="briefcase-blank"
        
    >
        **No, CinePro is for personal and non-commercial use only.** Selling this project or adding tracking/advertisements is strictly prohibited. For more details, refer to the [License](./license) section.
        By using this project, you agree to the terms outlined in the license.

        <Note>
            This project is not ment to be used for public sites or apps. Public usage will only lead to huge CPU and bandwidth usage and has a high chance of getting blocked by the providers.
        </Note>
    </Accordion>

    <Accordion
        title="Can I contribute to CinePro?"
        icon="code-pull-request"
        
    >
        **Yes!** **CinePro is an open-source project**, and contributions are welcome. You can contribute by reporting issues, suggesting features, or submitting pull requests. Check out the [Development](./development) section for more information on how to get involved.
    </Accordion>

    <Accordion
        title="How can I report a bug or request a feature?"
        icon="bug"
        
    >
        Please check if the issue is already reported on the [GitHub Issues page](https://github.com/cinepro-org/backend/issues). If not, feel free to [create a new issue](https://github.com/cinepro-org/backend/issues/new) with a detailed description of the bug or feature request. We appreciate your feedback and suggestions!
    </Accordion>

</AccordionGroup>

I hope this helps! If you have any more questions, feel free to [ask](https://github.com/orgs/cinepro-org/discussions).
