# Introduction

Welcome to the **CinePro Backend** documentation! This project is an open-source movie and TV show scraper API designed to provide links to media content available on the internet.

## Features

- **No Ads or Tracking**: Completely free of ads and tracking.
- **Supported Media Types**:
  - Movies
  - TV Shows
- **Caching**: Uses MongoDB to cache scraped data for better performance.

<Note>
  This project is for educational purposes only. It does not host or upload any media files.
</Note>
