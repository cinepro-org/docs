<Tabs>
  <Tab title="Windows" icon="windows">
    <Steps>
      <Step title="Install Node" icon="square-terminal">
        Make sure you have the following installed on your machine:
        <AccordionGroup>
          <Accordion icon="rectangle-terminal" title="Install Node.js">
            Install Node.js (v18 or higher) on your machine. You can download it from the
            [official website](https://nodejs.org/en/download/) or use winget to install it:
            `winget install OpenJS.NodeJS`.
          </Accordion>
        </AccordionGroup>
      </Step>

      <Step title="Clone the repository" icon="clone">
        Clone the CinePro repository to your local machine using Git. Open your terminal and run the following command:

        ```bash
        git clone https://github.com/cinepro-org/backend.git
        ```
      </Step>

      <Step title="Install dependencies" icon="cube">
        Navigate to the cloned repository directory and install the required dependencies using npm:

        ```bash
        cd backend
        npm i
        ```
      </Step>

      <Step title="Configure environment variables" icon="lock">
        Rename the `.env_example` file to `.env` and update the environment variables as needed. The `.env` file contains configuration settings for the CinePro application like API keys and debug settings.

      ```bash Example .env file
TMDB_API_KEY="your key here" # Replace with your TMDB API key. You can get one at https://www.themoviedb.org/settings/api
PORT="3000" # The port on which the server will run. You can change this to any available port. default is 3000
ALLOWED_ORIGINS=[] # Add allowed origins here. Example ["http://localhost:3001", "https://cinepro.mintlify.app"]
      ```

        <Info>
          CinePro requires a [TMDB](https://www.themoviedb.org/) API key to function properly. You can obtain a free API key by signing up on their website. Once you have the API key, add it to the `.env` file as `TMDB_API_KEY`.
        </Info>

        <Info>
          The `PORT` variable is the port on which the server will run. You can change it to any available port if needed.
        </Info>

        <Warning>
          Make sure to switch `DEBUG` to `off` in the `.env` file for production use. This will disable debug logging. To enable debug logging, set `DEBUG` to `true` or `1`.
        </Warning>
      </Step>

      <Step title="Start the server" icon="play">
        Run the following command to start the CinePro server:

        ```bash
        npm run main
        ```

        This will start the server, and you should see output indicating that the server is running.
      </Step>
    </Steps>
  </Tab>
